## ğŸ› ï¸ TROUBLESHOOTING QUICK FIXES

| Problem | Solution |
|---------|----------|
| Port 5000 in use | `lsof -i :5000` then `kill -9 PID` |
| npm install errors | `rm -rf node_modules && npm install --production` |
| Database connection failed | Check .env DATABASE_URL |
| PM2 not starting | `pm2 start server.js --name "pawnflow-api"` |
| API not responding | Check with `pm2 logs pawnflow-api` |
| SSL not working | Force HTTPS, clear cache, test at ssllabs.com |
| CORS errors | Check server.js CORS configuration |
| PDF not generating | Check /pdfs folder permissions |
